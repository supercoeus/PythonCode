# Python 入门示例

本目录包含若干简单的 Python 入门程序示例，每个文件都可以独立运行（不需要额外依赖）。

示例清单：

- `hello.py` - 输出 "Hello, world!"
- `calculator.py` - 简单演示加减乘除函数
- `fibonacci.py` - 打印前 10 个斐波那契数
- `file_io.py` - 简单的文件写入和读取示例
- `guess_number.py` - 非交互的猜数模拟（自动猜中）

如何运行（在 PowerShell 中）：

```powershell
python .\hello.py
python .\calculator.py
python .\fibonacci.py
python .\file_io.py
python .\guess_number.py
```
